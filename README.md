# hoodie-account

> Hoodie's Account API

## Local setup and tests

```
git clone git@github.com:hoodiehq/hoodie-store.git
cd hoodie-store
npm install
npm test
```

## Demo

Run the server:
```
npm start
```

After that go to `http://localhost::4663` in your browser.
